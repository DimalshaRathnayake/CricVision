CricVision Research Static Website
=================================

Project: PLAYER BASED SEGMENTATION AND WICKET EVENT RELATED VIDEO ANALYSIS IN CRICKET VIDEOS
Project ID: 25-26J-148
Website name: CricVision Research

How to run:
1. Extract this ZIP file.
2. Open index.html in a browser.
3. Keep the folder structure unchanged when uploading to course web.

Important:
- PP1 and PP2 PPTX files were not bundled because the uploaded PPTX files are very large and exceed the 20 MB course-web disk-space guideline.
- To add team photos, place images inside assets/images/team/ and replace the avatar placeholder in about.html.
- To add pending documents, place files inside assets/documents/ and update documents.html links.
